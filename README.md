# Terraform Azure AKS Infrastructure Project

## Overview
Provisioned Azure infrastructure using Terraform.

## Resources Created
- Azure Resource Group
- Azure Virtual Network (VNet)
- Azure Subnet
- Azure Container Registry (ACR)
- Azure Kubernetes Service (AKS)
- Azure Storage Account
- Azure Storage Container

## Terraform Concepts Used
- Providers
- Resources
- Variables
- Outputs
- Terraform State
- Remote Backend Foundation

## Commands Used

terraform init

terraform plan

terraform apply

terraform destroy
